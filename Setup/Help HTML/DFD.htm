Diagrama de Fluxo de Dados

Neste item, acontece a edição dos diagramas de fluxo de dados do modelo ambiental em vários níveis de detalhamento. É permitido criar uma nova visão para cada processo que se deseja detalhar.

As principais funções são:

- Imprimir : Imprime o diagrana

- Selecionar : Seleciona um elemento gráfico

- Editar : Permite a edição de propriedades do elemento gráfico selecionado

- Excluir : Exclui um elemento gráfico do diagrama

- Processo : Insere uma representação de um processo através da interface "Editar Processo".

- Entidade: Insere uma representação de um entidade através da interface "Selecionar Entidade".

- Depósito: Insere uma representação de um depósito através da interface "Editar Depósito".

- Fluxo: Insere uma representação de um fluxo de dados através da interface "Editar fluxo".

- Decompor : Abre uma nova janela com uma nova visão do processo selecionado para que seja possível detalhá-lo.

  • Verificar: Verfica a consistência do diagrama.
