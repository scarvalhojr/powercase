Criando e Editando Propriedades de um Projeto

No menu arquivo, escolha Novo. Uma janela aparecerá com campos em branco.Para se criar um novo projeto é necessário preencher todos estes campos, abaixo listados. A edição para os campos é normal e restringida somente aos campos Arquivo do Projeto, Criação e Última revisão.

-Nome Abreviado (título)

-Nome descritivo do projeto

-Responsável

-Data de Criação ( criada automaticamente pelo PowerCase)

-Última revisão ( criada automaticamente pelo PowerCase)

-Declaração de Objetivos
