Entidades

Entidades são os elementos do ambiente que irão interagir com o sistema .

Ao clicar no botão "Entidade" ou indo ao item Entidade do menu Modelo, uma janela com uma lista de entidades pré-cadastradas será mostrada, para que uma entidade possa ser selecionada. No caso de inserir uma nova entidade, nesta mesma janela, clica-se no botão "Inserir" para fornecer o nome da nova entidade.
