Lista de Entidades

Neste item, uma lista com todas as entidades já existentes é mostrada. Qualquer tipo de edição da lista ou de seus componentes é feita através dos botões localizados à sua direita ( Inserir, Editar, Excluir). Tanto o botão Inserir como o botão Editar abrem uma janela de diálogo para edição. Já o botão Excluir retira da lista o elemento selecionado.
